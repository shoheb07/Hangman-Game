# Hangman-Game
Hangman Game using HTML, CSS, and JavaScript. It includes a word display, keyboard input, and basic game logic.

A simple browser-based Hangman game where the player guesses letters to reveal a hidden word. The player has limited attempts before losing.

Features:
Random word selection
Tracks correct and wrong guesses
Displays remaining attempts
Win/Lose messages

How to Run:
Create three files: index.html, style.css, script.js
Copy the respective code into each file
Open index.html in a browser
